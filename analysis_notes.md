# TCGPlay 功能分析与全新概念提炼

## 现有模块概览

### 2.1 基础系统
- 2.1.1 Privy 登录与钱包连接
- 2.1.2 多语言支持系统（中英日韩）
- 2.1.3 个人用户中心

### 2.2 卡库系统（My Collection）
- 2.2.1 卡牌灵感空间（Card Mood Space）—— 3D展示间
- 2.2.2 卡牌故事（Card Story）—— 故事溯源
- 2.2.3 卡牌图集（Card Album）—— Cream Kawaii风格组合照
- 2.2.4 个人卡牌年度/交易总结（Annual Card Summary）

### 2.3 实时卡牌展览厅（Marketplace Gallery）
- 一行6张卡牌网格布局
- 左下角平台官方标识
- 悬停放大+光斑呼吸特效

## 需要优化为全新概念的方向

### 核心差异化概念提炼：

1. **卡牌灵感空间** → 可升级为"Card Soul Room"（卡魂空间）
   - 不仅是3D展示，而是卡牌与持有者的"灵魂绑定"概念
   - 引入"情感指数"——根据持有时长、交易频率、社区互动生成独特的灵魂共鸣值

2. **卡牌故事** → 可升级为"Card Chronicle"（卡牌编年史）
   - 不仅追溯历史，而是自动生成"卡牌传记"
   - 引入"传承链"概念——每位持有者可留下一句话，形成卡牌的"精神遗产"

3. **卡牌图集** → 可升级为"Card Constellation"（卡牌星座图）
   - 不仅是组合照，而是根据卡牌属性自动生成"星座关系图"
   - 展示卡牌之间的隐藏关联

4. **年度总结** → 可升级为"Collector DNA"（收藏家DNA）
   - 不仅是统计，而是生成独特的"收藏家人格画像"
   - 引入"收藏风格基因"概念

5. **展览厅** → 可升级为"Card Pulse"（卡牌脉搏）
   - 不仅是浏览，而是实时展示市场"心跳"
   - 引入"热度脉搏"视觉效果

## 自有概念核心：一行6张卡牌网格 + 左下角Renaiss标识
